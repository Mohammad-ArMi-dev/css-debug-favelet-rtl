Option 1: Use Pre-made Favelet

  1. Drag the CSS Debug Favelet link to your bookmarks bar
  2. Navigate to any webpage
  3. Click the bookmark to apply debugging styles

Option 2: Create Your Own

  1. Get the CSS file from: debug.css
  2. Go to Meyerweb Favelet Creator
  3. Paste the CSS URL in "URL of stylesheet"
  4. Name it (e.g., "CSS Debug") and create the favelet
  5. Drag to your bookmarks bar

  • All elements: Dashed red outline
  • Tables: Solid red border
  • Table cells: Dashed purple border
  • Inline styles: Orange dotted border
  • Images without alt: Solid red border
  • Font tags: Yellow background
  • Empty links: Blue outline
  • RTL elements: Light blue background
  • LTR elements: Light pink background
